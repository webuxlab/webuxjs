## Introduction

This module offers two components:

- Custom telemetry service
- Jaeger Tracing through Open Telemetry

## Installation

```bash
npm install --save @studiowebux/telemetry
```

[NPM](https://www.npmjs.com/package/@studiowebux/telemetry)
